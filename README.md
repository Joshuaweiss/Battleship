# Simple Battleship
===================

## Install

### Browser (Not needed the compiled assets are checked in)

1. ```cd browser```
2. ```npm install```
3. ```typings install```
4. ```npm run build```

### Server

1. ```cd server```
2. ```bundle install```
3. ```rails s```


## Develop

developed with webpack dev server

### Browser

* ```npm start```

### Server

* ```USE_DEV_SERVER=true rails s```
